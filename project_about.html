Описание

Бизнес центр «Ленинский проспект, 119» представляет собой уникальный проект, совмещающий деловую атмосферу офиса с комфортом зеленой зоны парка. Это новое здание класса А, спроектированное с учетом самых строгих требований современного бизнеса, обладает следующими ключевыми характеристиками:

  • Расположение в живописной парковой зоне;
  • Современная архитектура с элементами Hi-Tech;
  • Просторная входная группа, 3-х уровневый атриум;
  • Готовые к въезду помещения;
  • Высококачественная отделка, включающая разводку всех инженерных систем, установку фанкойлов, фальш-пол, ковролин;
  • Эффективные открытые пространства;
  • Гибкость планировочных решений;
  • Профессиональная управляющая компания;
  • Столовая для арендаторов расположена на -1 этаже.
